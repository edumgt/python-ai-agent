> 제안요청서 (Request for Proposal) | 📊 AI 기반 자동화 투자 시스템

---

## 1. 프로젝트 개요 (Project Overview)

| 항목 | 내용 |
|------|------|
| **프로젝트명** | 나만의 로보 어드바이저 개발 및 성과 검증 프로젝트 |
| **유형** | AI 기반 자동화 투자 시스템 구축 |
| **목표** | 패턴 인식 기법을 활용한 주식 시장 예측, 자산배분 모델 기반 포트폴리오 최적화, 주식 스크리닝을 통한 종목 선정, 퀀트 모델 결과 해석 및 모의 투자 의사결정 수행 |

## 2. 배경 및 목적 (Background & Purpose)

### 2.1 배경

- 금융 시장의 데이터 기반 의사결정 수요 증가
- AI·머신러닝 기술을 활용한 자동화 투자 시스템(로보 어드바이저)의 상용화 확산
- 개인 투자자의 퀀트 모델 접근성 향상 필요성 증대

### 2.2 목적

- **AI 기반 로보 어드바이저 모델 자체 개발** 및 성과 검증 역량 확보
- 주식 시장 데이터에서 **패턴 인식 기법**을 적용하여 예측 모델 구축
- **자산배분 모델(Asset Allocation Model)** 을 활용한 포트폴리오 최적화 실습
- **주식 스크리닝(Stock Screening)** 기법으로 투자 유망 종목 선별
- 구축한 퀀트 모델의 결과를 해석하고 **모의 투자 의사결정** 직접 수행

## 3. 요구사항 (Requirements)

### 3.1 기능 요구사항

#### 3.1.1 AI 기반 로보 어드바이저 모델 개발

- [ ] 머신러닝(ML) 및 딥러닝(DL) 기반 주식 가격 예측 모델 구현
  - 지도학습: RandomForest, XGBoost, LSTM, Transformer 등
  - 비지도학습: 군집화(Clustering) 기반 시장 국면 분류
- [ ] 모델 학습 파이프라인 구축 (데이터 수집 → 전처리 → 학습 → 검증)
- [ ] 모델 성능 지표 정의 및 측정 (Accuracy, F1-Score, Sharpe Ratio 등)

#### 3.1.2 패턴 인식 기법을 활용한 주식 시장 예측

- [ ] 기술적 패턴 인식: Head & Shoulders, Double Top/Bottom, 삼각형 패턴 등
- [ ] 시계열 패턴 분석: 자기상관(ACF/PACF), ARIMA, Fourier Transform
- [ ] 딥러닝 기반 패턴 인식: CNN을 활용한 차트 이미지 분류, LSTM 시계열 예측
- [ ] 멀티팩터 신호(Multi-factor Signal) 통합 및 앙상블 모델 구성

#### 3.1.3 자산배분 모델을 활용한 포트폴리오 최적화

- [ ] 평균-분산 최적화(Mean-Variance Optimization, Markowitz) 구현
- [ ] 리스크 패리티(Risk Parity) 및 최소 분산(Minimum Variance) 포트폴리오 구성
- [ ] Black-Litterman 모델 적용 (투자자 뷰 반영)
- [ ] 동적 자산배분(Dynamic Asset Allocation): 시장 국면에 따른 비중 재조정

#### 3.1.4 주식 스크리닝을 통한 종목 선정

- [ ] 팩터 기반 스크리닝: 가치(Value), 모멘텀(Momentum), 퀄리티(Quality), 저변동성(Low Volatility)
- [ ] 재무제표 기반 필터링: PER, PBR, ROE, 부채비율 등 정량 기준 설정
- [ ] 기술적 스크리닝: 이동평균 정배열, 52주 신고가 근접 종목 등
- [ ] 스크리닝 결과 랭킹 시스템 구현 (복합 점수 산출)

### 3.2 비기능 요구사항

- [ ] 데이터 소스: KRX, Yahoo Finance, FinanceDataReader 등 공개 데이터 활용
- [ ] 구현 언어: Python 3.10 이상
- [ ] 재현 가능성: 코드 및 랜덤 시드(Seed) 고정, 결과 재현 보장
- [ ] 문서화: 모델 설계 문서, 분석 보고서, 코드 주석 포함

## 7. 평가 기준 (Evaluation Criteria)

### 7.1 모델 성능 평가

| 지표 | 기준값 | 비고 |
|------|--------|------|
| 예측 모델 정확도 | 방향성 정확도 55% 이상 | 상승/하락 방향 예측 |
| 포트폴리오 수익률 | 벤치마크(KOSPI) 대비 초과수익 | 백테스트 기준 |
| 최대낙폭 (MDD) | -20% 이내 | 리스크 관리 기준 |
| Sharpe Ratio | 1.0 이상 | 위험 대비 수익률 |
| 스크리닝 종목 수익률 | 시장 평균 대비 상위 30% | 팩터 유효성 검증 |

### 7.2 프로세스 평가

- [ ] 데이터 전처리 및 피처 엔지니어링의 합리성
- [ ] 모델 선택 근거 및 하이퍼파라미터 튜닝 과정의 체계성
- [ ] 퀀트 모델 결과에 대한 비판적 해석 능력
- [ ] 모의 투자 의사결정의 논리적 일관성
