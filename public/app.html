📚 주요 인디케이터 개념

          

            

              
이동평균 (MA)

              
일정 기간 종가의 평균. MA5 > MA20 교차 시 골든크로스(매수 신호), MA5 < MA20 교차 시 데드크로스(매도 신호).

            

            

              
RSI (상대강도지수)

              
14일 기준 상승·하락 강도 비율. 70 이상은 과매수(매도 주의), 30 이하는 과매도(매수 기회) 구간.

            

            

              
MACD

              
12일 EMA - 26일 EMA. 시그널선(9일 EMA) 상향 돌파 시 매수, 하향 돌파 시 매도 신호.

📖 모델별 특성

        

          
SVM (Linear/RBF)
마진 최대화 분류. 고차원 피처에 강함. RBF 커널은 비선형 결정경계.

          
RandomForest
여러 의사결정트리의 배깅 앙상블. 과적합 방지, 피처 중요도 제공.

          
GradientBoosting
잔차를 순차 학습하는 부스팅 앙상블. 높은 정확도, 느린 학습.

          
MLP (신경망)
다층 퍼셉트론. 64→32 은닉층, Adam 옵티마이저, 조기 종료.

          
Ensemble (Voting)
RF+GB+MLP 다수결. 개별 모델 약점 보완, 안정적 성능.

          
LightGBM
Leaf-wise 트리 성장. 빠른 학습, 높은 정확도. 대용량 데이터 적합.

📖 회귀 모델 개요

        

          
Linear Regression
최소제곱법으로 선형 관계 추정. 해석이 쉬우나 과적합 위험.

          
Ridge (L2 정규화)
가중치 제곱합 페널티로 과적합 방지. α 클수록 계수 축소.

          
Lasso (L1 정규화)
불필요한 피처 계수를 0으로 수렴(피처 선택 효과).

          
SVR (RBF 커널)
ε-tube 내 오차를 허용하는 서포트벡터 회귀. 비선형 패턴 포착.

        

      

    


    
    

      

        
 종목 군집화 (KMeans)

        
퀀트 대표 종목의 기술적 피처를 KMeans 클러스터링으로 군집화합니다.

        
        
데이터 수집 및 군집화 중...

        

          

          

            

              

                종목군집
                RSIMA5 비율
                변동성(BB)거래량비
              

              
            

          

          

        

      

      

        
📖 클러스터링 기법

        

          
KMeans
사전에 K 개수 지정. 유클리드 거리 기반 중심점 최적화. 속도 빠름.

          
계층적 군집화
거리 행렬로 계층 구조 형성. 덴드로그램으로 최적 K 파악 가능.

          
DBSCAN
밀도 기반 군집화. 노이즈 포인트 처리. 비구형 군집 탐지.

📖 GridSearchCV 개요

        
GridSearchCV는 지정된 하이퍼파라미터 격자(Grid)를 완전 탐색하여 교차검증 기준 최적 조합을 찾습니다. RandomizedSearchCV는 무작위 샘플링으로 더 큰 탐색 공간을 빠르게 탐색합니다. 베이즈 최적화(Optuna, Hyperopt)는 이전 결과를 활용해 효율적으로 수렴합니다.

실습 환경 (PyTorch/Keras)

        
# PyTorch LSTM 예시 (Google Colab / Jupyter)
import torch, torch.nn as nn

class StockLSTM(nn.Module):
    def __init__(self, input_size=12, hidden=128, layers=2, n_class=3):
        super().__init__()
        self.lstm = nn.LSTM(input_size, hidden, layers,
                            batch_first=True, dropout=0.2)
        self.fc = nn.Linear(hidden, n_class)

    def forward(self, x):          # x: (batch, seq, features)
        out, _ = self.lstm(x)
        return self.fc(out[:, -1]) # 마지막 타임스텝

model = StockLSTM()
optimizer = torch.optim.Adam(model.parameters(), lr=1e-3)
criterion = nn.CrossEntropyLoss()

📖 산업 분석 방법론

        

          
포터의 5가지 힘
공급자/구매자 교섭력, 신규진입 위협, 대체재, 내부 경쟁으로 산업 매력도 평가.

          
산업 수명주기
도입→성장→성숙→쇠퇴. 성장기 = 높은 PEG, 성숙기 = 안정 배당, 쇠퇴기 = 가치 덫.

        

      

    


    
    

      

        
 재무제표 분석 (DCF·EVA·FCF)

        

          
          
        

        

          

        

      

      

        
📖 재무 분석 3대 방법론

        

          

            DCF (할인현금흐름)

            
V = Σ FCFt / (1+r)^t
            + TV / (1+r)^n

            미래 FCF를 WACC로 할인해 내재가치 산출. 성장률·할인율에 민감.
          

          

            EVA (경제적 부가가치)

            
EVA = NOPAT - WACC × 투하자본

            자본비용 초과 수익 여부 측정. EVA > 0이면 가치 창출.
          

          

            FCF (잉여현금흐름)

            
FCF = 영업CF - CapEx

            실제 배당·투자 가능 현금. FCF 수익률 = FCF / 시가총액.
          

        

      

    


    
    

      

        
 기술적 분석

        

          

            
캔들 패턴

            

              
● 망치형 (Hammer) — 하락 추세 반전 신호. 아래 꼬리가 몸통의 2배 이상.

              
● 유성형 (Shooting Star) — 상승 추세 반전. 위 꼬리가 몸통의 2배.

              
● 강세 장악형 (Bullish Engulfing) — 이전 음봉을 완전히 포함하는 양봉.

              
● 도지 (Doji) — 시가≈종가. 추세 불확실성 신호.

              
● 아침 별 (Morning Star) — 3캔들 패턴. 하락→도지→강한 양봉.

            

          

          

            
엘리어트 파동이론

            
상승 5파 + 조정 3파 = 완전 사이클

 5
3         B
  1    A     C
    2
  4

1·3·5파: 충격파 (추세 방향)
2·4파:   조정파
A·B·C파: 조정 사이클

            
황금비율(0.618·1.618)로 파동 목표가·조정 깊이 계산.

          

        

      

      

        
주요 지지/저항 도구

        

          
피보나치 되돌림
고점→저점 이후 38.2%, 50%, 61.8% 수준이 주요 지지·저항.

          
이동평균선 정배열
MA20 > MA60 > MA120 = 강세 추세. 역배열 = 약세.

거래량 동반 돌파 = 신뢰도↑. 거래량 없는 돌파 = 페이크아웃 주의.

        

      

    


    

    
    

      

        
 금융상품 이해

        

          

            
주식 (Equity)

            

              
• 보통주: 의결권 + 배당권. 청산 시 후순위.

              
• 우선주: 배당 우선, 의결권 없음. 채권과 주식 중간.

              
• PER = 주가/EPS, PBR = 주가/BPS, PSR = 시총/매출

              
• 배당수익률 = DPS/주가 × 100

            

          

          

            
ETF (상장지수펀드)

            

              
• 주식처럼 거래되는 펀드. NAV 추적.

              
• 패시브(지수 추종) vs 액티브 ETF

              
• 레버리지 2x/3x: 일별 수익률 배수 (장기 복리 손실 주의)

              
• 인버스: 지수 하락 시 수익. 헤지 수단.

            

          

          

            
채권 (Bond)

            

              
• 금리↑ → 채권 가격↓ (역관계)

              
• 듀레이션: 금리 민감도. D년 채권은 금리 1%↑ → 가격 D%↓

              
• 신용등급: AAA(무위험) → CCC(투기). 스프레드 = 위험 프리미엄.

              
• YTM (만기수익률): 현재 가격 기준 보유 수익률.

            

          

          

            
파생상품

            

              
• 선물(Futures): 미래 특정 가격에 매매 약정. 헤지·투기.

              
• 옵션(Options): 콜(매수권) / 풋(매도권). 프리미엄 지급.

              
• 스왑(Swap): 금리·통화 교환. 기관 헤지 도구.

              
• 그릭스: Delta(방향) · Gamma(볼록성) · Theta(시간감소) · Vega(변동성).

            

          

        

      

    


    
    

      

        
 자산배분 모델

        

          

            
평균-분산 최적화 (MVO)

            
min w'Σw  s.t. w'μ = r_target
                     Σwi = 1, wi ≥ 0

효율적 프론티어: 동일 기대수익 최소 분산.
샤프 비율 최대화: 접선 포트폴리오.

            
단점: 기대수익 추정 오차에 민감, 극단 배분 발생.

          

          

            
블랙-리터만 (Black-Litterman)

            
μ_BL = [(τΣ)^-1 + P'Ω^-1P]^-1
       × [(τΣ)^-1 Π + P'Ω^-1 Q]

Π: 시장 균형 수익률 (역최적화)
P: 투자자 뷰 행렬
Q: 뷰 기대수익
Ω: 뷰 불확실성

            
투자자 뷰를 시장 균형에 베이즈 방식으로 결합.

          

          

            
Risk Parity

            
목표: 각 자산의 위험 기여도 균등화.

RC_i = wi × (Σw)_i / √(w'Σw)

모든 i에 대해 RC_i = 1/n 되도록 w 최적화.

            
레버리지 필요. 저변동성 자산(채권) 비중↑.
